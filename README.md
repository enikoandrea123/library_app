# Library Application


The Library Application is a simple system to manage books and users within a library. It provides features for borrowing and returning books, viewing book catalogs, managing user data, and searching for books. The project is designed with a user-friendly interface that can be accessed through a web browser.
http://localhost:8080/

## Technologies

•	Spring Boot

•	Thymeleaf

•	Spring Data JPA

•	H2 Database

•	HTML/CSS/JavaScript

•   Spring Security

•	BCrypt Password Encoder
